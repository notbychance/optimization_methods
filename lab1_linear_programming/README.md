# Лабораторная работа №1. Решение задач линейного программирования

Модуль решает двумерные задачи линейного программирования графическим методом:

1. читает модель из JSON;
2. строит ограничения и область допустимых решений;
3. находит угловые точки;
4. вычисляет значение целевой функции в каждой угловой точке;
5. выбирает оптимальную точку;
6. сохраняет Markdown-отчет и PNG-график.

Для варианта 17 исходная задача содержит три изделия: телевизоры, стереосистемы и акустические системы. Условие «спрос на стереосистемы в два раза выше, чем на акустические системы» используется как связь `x2 = 2y`, поэтому задача сводится к двум переменным:

- `x1` — количество телевизоров;
- `y` — количество акустических систем;
- `x2 = 2y` — количество стереосистем.

## Запуск

Из корня проекта:

```bash
python main.py -n 1 -i lab1_linear_programming/examples/lab1_variant17.json -s results/lab1_variant17.md
```

Или из каталога лабораторной:

```bash
python main.py -i examples/lab1_variant17.json -s ../results/lab1_variant17.md
```

После запуска будут созданы:

- `results/lab1_variant17.md` — отчет;
- `results/lab1_variant17.png` — графическая интерпретация решения.
